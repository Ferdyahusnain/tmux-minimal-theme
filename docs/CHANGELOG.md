# Changelog

## [1.0.0] - 2025-08-30

### Added
- Initial release of tmux-minimal-theme
- Support for multiple color schemes (Catppuccin, Tokyo Night, Dracula, Gruvbox)
- System monitoring (CPU, memory, battery, time)
- Customizable colors and components
- TPM installation support
- Comprehensive documentation

### Features
- Clean minimal design
- Session name display with icon
- Current directory display
- Memory usage percentage
- Date and time display
- Battery percentage (Linux systems)
- Customizable color schemes
- Vim-like copy mode styling
